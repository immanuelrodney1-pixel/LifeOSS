PROJECT NAME
LifeOS — Personal Life Tracking System

PROJECT TYPE
Mobile-first web application

PURPOSE
LifeOS is a deeply personal life tracking system designed to allow users to record everything they do throughout the day in structured time blocks while also tracking emotional states and thought patterns.

The system combines:

• time tracking
• emotional journaling
• pattern recognition
• personal analytics

The goal is to allow a user to understand how their time, attention, and emotional state interact.

PRIMARY USE CASE
The user logs activities throughout the day in 5-minute precision time blocks.

Example:

08:00–08:45 Deep Work
08:45–09:00 Email
09:00–09:30 Meeting

At the same time the user records emotional reflections including:

• mood
• thought patterns
• emotional intensity
• notes

The dashboard continuously updates to show how the day is unfolding.

TARGET USER
Initially: single personal user

Future:
• productivity enthusiasts
• founders
• high performers
• journaling users

PLATFORM
Web application
Mobile-first responsive design.

KEY PRINCIPLES

1. EXTREME CLARITY
The interface should feel calm and simple.

2. FAST INPUT
Logging an activity should take under 3 seconds.

3. HIGH SIGNAL
Dashboards must only show meaningful information.

4. REFLECTIVE
The app should help the user understand patterns.

CORE FEATURES

ACTIVITY TRACKING
Log activity blocks with start and end times.

EMOTION JOURNAL
Log moods, thoughts, and reflections.

DAY DASHBOARD
See live totals of time spent.

WEEK REVIEW
Review how the week unfolded.

PATTERN INSIGHTS
See repeated thoughts and dominant activities.
